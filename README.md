# DataForecasting

An end-to-end financial data forecasting tool that automatically collects market data, performs technical trend analysis, and predicts future stock prices using machine learning.

## Features

- **Automated Data Collection:**  
  Fetches stock and crypto data from Yahoo Finance using user-specified tickers.

- **Feature Engineering:**  
  Generates additional technical indicators such as moving averages, Fibonacci retracement levels, and support/resistance zones.

- **Exploratory Data Analysis (EDA):**  
  Visualizes price trends, volume changes, and key support/resistance levels.

- **Machine Learning Forecasting:**  
  Uses a trained Linear Regression model to predict future prices for user-defined days.

- **Visualization:**  
  Displays extended charts that overlay predictions as dotted extensions of the original trend, including moving averages and Fibonacci levels.
